# SQL Analytics Projects

This repository contains SQL-based analytics projects focused on answering real-world business questions using transactional data.

## Objective
Demonstrate practical SQL skills used by Data Analysts and Business Analysts, including data aggregation, joins, KPI analysis, and business insights.

## Tools
- SQL
- PostgreSQL / MySQL (syntax-based)
- Excel (optional)

## Projects
- Exploratory revenue analysis
- Customer behavior analysis
- KPI reporting using SQL

## Next Steps
- Add datasets and structured SQL analysis
- Expand to advanced SQL (CTEs, window functions)

## Schema
| Column Name        | Data Type        | Description                                |
| ------------------ | ---------------- | ------------------------------------------ |
| `order_id`         | INTEGER / UUID   | Unique identifier for each order           |
| `order_date`       | DATE / TIMESTAMP | Date when the order was placed             |
| `customer_id`      | INTEGER / UUID   | Unique identifier for each customer        |
| `product_category` | TEXT / VARCHAR   | Product category associated with the order |
| `quantity`         | INTEGER          | Number of units purchased                  |
| `revenue`          | NUMERIC / FLOAT  | Total revenue generated by the order       |
